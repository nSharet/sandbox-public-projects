# Feature Expert Agents

Feature Expert Agents is a starter project inside `sandbox-public-projects` for defining and organizing AI agents that specialize in feature discovery, analysis, design review, implementation planning, and code review.

## Goals
- Create a clear home for feature-focused expert agents
- Define responsibilities, boundaries, and collaboration flow between agents
- Provide reusable prompts, examples, and documentation
- Support future experimentation with agent-driven product and engineering workflows

## Initial Structure
- `docs/` – project vision, architecture, roadmap
- `agents/` – agent role definitions
- `examples/` – example feature requests and expected workflows
- `prompts/` – system prompts and evaluation checklists

## Planned Agents
1. Feature Analyst
2. Design Reviewer
3. Implementation Planner
4. Code Reviewer

# Sandbox Public Projects

A collection of proof-of-concept (POC) projects and solutions focused on **design and architecture challenges**.

## Purpose

This repository serves as a sandbox for:
- Exploring and solving general software engineering challenges
- Creating proof-of-concept implementations
- Experimenting with design patterns and architectural approaches
- Demonstrating best practices in software design

## Structure

Each folder in this repository represents a **separate, standalone project** with:
- Its own problem definition
- Specific goals and targets to achieve
- Independent implementation and documentation

## How to Navigate

Browse the folders to find specific challenges. Each project folder contains:
- A dedicated README explaining the challenge
- The target solution or POC implementation
- Any relevant documentation or diagrams

## Topics Covered

Projects in this repository may include challenges related to:
- Design Patterns (GoF patterns, architectural patterns)
- SOLID Principles
- Clean Architecture
- Microservices Design
- Event-Driven Architecture
- Domain-Driven Design (DDD)
- Scalability & Performance
- Other software design topics

## Contributing

Feel free to explore, learn, and contribute new challenges or improvements to existing solutions.
